# neighborhood props lexicon

Use this file when generating or editing **equipment, props, item icons, environment-aware standees, or backgrounds with neighborhood context**. Treat it as a structured vocabulary of grounded old-community objects.

## usage rules

- Pick props that support the subject's story instead of filling space randomly.
- Prefer **2 to 5 coherent object groups** over many isolated objects.
- Use objects with believable placement: storefront edge, doorway side, curbside, stall tabletop, bicycle area, tree base, wall corner.
- Keep prop forms readable. Simplify labels, logos, and printed text.
- Favor worn-but-cared-for objects rather than pristine showroom objects or post-apocalyptic debris.
- For direct-use item icons, isolate the object and do not add neighborhood context unless the user explicitly wants a scene-style prop.

## storefront and street-edge categories

### small shop exteriors
- half-open rolling shutter
- simple glass door
- tiled lower wall
- faded painted signboard
- hanging shop lamp
- narrow awning
- doorway curtain strip
- stacked plastic crates near entrance
- handwritten price board
- old menu board without dense text

- barber pole simplified and muted
- salon mirror glow from inside
- waiting stool by the entrance
- broom and dustpan set
- small potted plant
- towel rack implied in silhouette

### corner store and snack counter cues
- drink crate stack
- freezer chest silhouette
- snack shelf block shapes
- small hanging sign
- plastic stool pair
- umbrella bucket near door
- boxed goods by wall

## environment micro-clutter formulas

Use these as grouped rhythms.

- plastic stool + flower pot + broom + utility box
- crate stack + handwritten price card + basin + hanging light
- bicycle + curb rail + tree base + fallen leaves
- doorway shoe rack + umbrella stand + folded chair
- scooter + delivery box + wall poster + potted plant

## icon-ready prop extraction

When the request is for a **single usable game prop or item icon**, reduce the object to one centered subject and keep only the most identity-bearing attachments.

Examples:
- market basket icon -> woven basket with leafy greens only, no stall background
- barber item icon -> compact muted barber pole or scissors-only icon, no storefront
- old-community utility prop -> meter box or wall lamp only, isolated on transparent background
- curbside object icon -> plastic stool or flower pot only, centered and cleanly framed

## forbidden drift for props and backgrounds

Default negative cues when relevant:
- futuristic sci-fi kiosks
- sleek luxury storefronts
- glossy shopping-mall finishes
- overgrown ruin aesthetics
- trash chaos everywhere
- hyper-detailed text blocks
- western suburban garage cues dominating the scene
- ornate fantasy market props unless requested
- cyberpunk neon signage
- showroom product photography styling
