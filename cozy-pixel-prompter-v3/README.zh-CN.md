# Cozy Pixel Prompter

[English](./README.md)

一个用于生成像素风游戏资产生成提示词的技能包，专注于柔和现代东亚社区家庭风格。

## 功能特性

- **风格一致性**：在所有资产中保持统一的治愈系现代像素美学
- **多资产类型**：支持立绘、精灵图、装备、道具、物品和小头像
- **生产就绪**：生成干净、可直接用于游戏流程的资产（透明背景）
- **参考图编辑**：保留原始风格的同时应用用户要求的修改
- **角色差异化**：确保不同职业/角色具有明显区分度
- **双语支持**：英文输出为主，可选中文注释

## 使用方法

本技能专为 OpenClaw agent 设计。当用户请求以下内容时触发：

- 单个角色、站立精灵或立绘
- 装备、道具或物品
- 小头像或图标

技能会生成完整的提示词包，包括：

1. **风格对齐**：说明请求如何适配家庭风格
2. **正向提示词**：可直接用于生成的提示词
3. **负向提示词**：防止风格漂移的约束
4. **资产规格**：画布尺寸、构图、对齐方式等
5. **可选模型备注**：模型特定建议

## 风格概览

**核心美学**：治愈系现代东亚社区像素风

- 柔和低饱和色彩，局部暖色点缀
- 克制 Q 版比例（约 2-3 头身）
- 干净像素簇，柔和阴影
- 日常生活切片主题
- 有生活痕迹的老社区真实感

## 系统要求

- OpenClaw（支持 skill 功能）
- 图像生成模型（推荐：DALL-E、Midjourney 或类似模型）
