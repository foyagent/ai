# character clothing lexicon

Use this file when generating or editing **characters, standees, sprites, or small avatars**. Treat it as a controlled vocabulary for clothing and carry items inside the family style.

## usage rules

- Build outfits from **1 silhouette anchor + 1 lower-body anchor + 1 to 2 garment anchors + 0 to 2 accessory anchors**.
- For profession-driven characters, combine this file with `character-differentiation.md`; do not solve identity with clothing alone.
- Prefer everyday clothing that feels believable in a lived-in neighborhood.
- Keep forms readable at pixel scale. Favor broad garment shapes over many tiny details.
- Let hairstyle, outer silhouette, lower-body structure, and one key accessory carry identity.
- Avoid runway styling, luxury branding, tactical gear, ornate fantasy layering, and excessive jewelry unless the user explicitly asks.
- For sprites and avatars, simplify small seams, buttons, and prints aggressively.
- When editing from a reference, preserve the original outfit structure unless the user explicitly asks to replace it.
- Do **not** let every outfit collapse into cream, khaki, and yellow-brown. Use a muted family palette, but still choose a distinct dominant clothing color family.
- Do **not** default to straight-leg pants unless the user's reference or role clearly calls for them.

## forbidden drift for outfits

Default negative cues when relevant:
- monochrome yellow-beige outfit for every role
- same cream shirt plus khaki straight pants on every character
- luxury fashion styling
- runway silhouette
- tactical straps everywhere
- glossy leather shine
- heavy armor plating
- magical ornament overload
- oversized weapons attached to clothing
- neon trims
- brand-logo heavy streetwear
- maid costume, idol costume, or cosplay styling unless requested

## anti-repetition reminders

When multiple different roles are requested across one session, avoid repeating the same default combination of:
- cream collared shirt
- brown or khaki straight pants
- black canvas sneakers
- front-facing neutral stand
- slim young-adult body

Break repetition by changing **dominant color family**, garment structure, lower-body structure, footwear, age cue, stance, and carry item together.
