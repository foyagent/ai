# AI

Agent skills & resources for OpenClaw.

## Skills

### auto-e2e

浏览器自动化录制/回放工具。通过自然语言描述操作步骤，自动生成可复用的 Playwright 脚本。

**功能：**
- `/auto-e2e <url>` - 打开页面开始录制
- `/auto-e2e replay <query>` - 回放历史录制
- `/auto-e2e alias <name> <script>` - 为脚本绑定别名

**版本：** 1.4.0

详见 [skills/auto-e2e/SKILL.md](skills/auto-e2e/SKILL.md)
